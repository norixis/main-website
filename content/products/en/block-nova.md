# Block Nova

Block Nova is a fast, arcade-style puzzle game for iOS where players drag and match blocks under pressure, build combo streaks, and chase high scores across multiple modes and difficulties.

## Core Gameplay

- Drag the active block to match targets.
- Score rises with speed and accuracy.
- Combo streaks increase score multipliers.
- In timed modes, players race against countdown pressure.
- In Zen mode, there is no timer, focused on pure flow and long runs.

## Game Modes

- Classic Mode: balanced pace and progression.
- Precision Mode: exact color + shape matching for higher challenge.
- Zen Mode: no timer, combo-focused sessions.

## Difficulty System

- Easy: slower pace, more forgiving.
- Medium: balanced challenge.
- Hard: faster, stricter, higher reward potential.

## Retention and Competitive Features

- Game Center leaderboards (global + optional mode/difficulty boards).
- Achievement progression tied to score milestones.
- Mode+difficulty best score tracking.
- Run summary screen with score, mode best, and best combo.
- Bilingual support (English + Simplified Chinese).
